## Getting Started / 어떻게 시작하나요?

### Prerequisites / 선행 조건

- JDK 11
- Maven

### Build & Run / 빌드 및 실행

```bash
$ mvn package
$ java -jar target/studyolle-0.1.1-SNAPSHOT.jar
```

```bash
$ mvn spring-boot:run
```

* [http://localhost:8080](http://localhost:8080)
